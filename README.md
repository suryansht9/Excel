## üßÆ 1. Common Formulas

| Formula            | Description                        | Example                  |
|--------------------|------------------------------------|--------------------------|
| `=A1+A2`           | Adds two cells                     | `=5+10` ‚Üí `15`           |
| `=A1-A2`           | Subtracts two cells                | `=20-5` ‚Üí `15`           |
| `=A1*A2`           | Multiplies two cells               | `=4*5` ‚Üí `20`            |
| `=A1/A2`           | Divides two cells                  | `=20/4` ‚Üí `5`            |
| `=A1^2`            | Exponentiation                     | `=5^2` ‚Üí `25`            |

## üìê 2. Useful Functions

| Function              | Description                                      |
|------------------------|--------------------------------------------------|
| `=SUM(A1:A5)`          | Adds values from A1 to A5                        |
| `=AVERAGE(A1:A5)`      | Calculates average                               |
| `=MAX(A1:A5)`          | Finds the largest number                         |
| `=MIN(A1:A5)`          | Finds the smallest number                        |
| `=IF(A1>50,"Pass","Fail")` | Logical condition (returns Pass or Fail)   |
| `=VLOOKUP(101, A2:C10, 2, FALSE)` | Searches a value in a table vertically |
| `=COUNTIF(A1:A10, ">60")` | Counts cells greater than 60                 |

## üé® 3. Formatting Tips

- **Bold, Italic, Underline**: Use toolbar or `Ctrl+B`, `Ctrl+I`, `Ctrl+U`
- **Cell Color**: Home ‚Üí Fill Color
- **Font Color**: Home ‚Üí Font Color
- **Number Format**: Right-click ‚Üí Format Cells ‚Üí Choose "Currency", "Percentage", etc.
- **Conditional Formatting**: Home ‚Üí Conditional Formatting ‚Üí Highlight Rules

---

## üìà 4. Creating Charts

1. Select your data
2. Go to **Insert > Charts**
3. Choose chart type (Column, Line, Pie, Bar)
4. Add chart title, axis labels, legends as needed
